# UBL 2.1 XSD Schemas

This directory contains the XML Schema Definition (XSD) files required for
validating UBL 2.1 electronic invoices generated by this module.

## Required Files

| File | Source | Purpose |
|------|--------|---------|
| `UBL-Invoice-2.1.xsd` | [OASIS UBL 2.1](https://docs.oasis-open.org/ubl/os-UBL-2.1/) | XSD validation for UBL Invoice documents |
| `UBL-CreditNote-2.1.xsd` | [OASIS UBL 2.1](https://docs.oasis-open.org/ubl/os-UBL-2.1/) | XSD validation for UBL CreditNote documents |
| `common/` | [OASIS UBL 2.1](https://docs.oasis-open.org/ubl/os-UBL-2.1/) | Common components (CBC, CAC, extension) |

## Download Instructions

1. Download the UBL 2.1 OS package from OASIS:
   https://docs.oasis-open.org/ubl/os-UBL-2.1/UBL-2.1.zip

2. Extract the following into this directory:
   - `xsd/maindoc/UBL-Invoice-2.1.xsd` → `UBL-Invoice-2.1.xsd`
   - `xsd/maindoc/UBL-CreditNote-2.1.xsd` → `UBL-CreditNote-2.1.xsd`
   - `xsd/common/` → `common/`

3. Verify the schemas are accessible by running module validation tests.
