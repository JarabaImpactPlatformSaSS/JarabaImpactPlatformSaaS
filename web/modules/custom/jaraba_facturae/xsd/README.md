# Facturae 3.2.2 XSD Schema

This directory should contain the official Facturae 3.2.2 XSD schema file.

## Required File

- `Facturaev3_2_2.xsd` — Official XML Schema Definition from the Spanish Government

## Source

Download from the official Facturae website:
https://www.facturae.gob.es/formato/Versiones/Facturaev3_2_2.xsd

## Policy File

The signing policy PDF should be placed in `../policy/`:
- `Facturaev3_2_2-policy.pdf` — Signature policy for XAdES-EPES signing

Download from:
http://www.facturae.gob.es/politica_de_firma_formato_facturae/politica_de_firma_formato_facturae_v3_1.pdf
