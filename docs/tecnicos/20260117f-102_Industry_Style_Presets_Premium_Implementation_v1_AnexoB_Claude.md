1. Resumen de Verticales Institucionales
Este anexo complementa el documento principal de Industry Style Presets con los verticales institucionales y formativos del Ecosistema Jaraba. Estos verticales tienen requisitos específicos de compliance, accesibilidad y tono institucional.
1.1 Diferencias con Verticales Comerciales
Aspecto	Comerciales	Institucionales
Tono visual	Diferenciación, personalidad de marca	Confianza, seriedad, accesibilidad
Colores	Paletas expresivas, identidad única	Paletas institucionales, alto contraste
Tipografía	Serifs elegantes, display fonts	Sans-serif legibles, tamaños generosos
Accesibilidad	WCAG 2.1 AA recomendado	WCAG 2.1 AA obligatorio
Animaciones	Expresivas, premium	Sutiles, reducibles (prefers-reduced-motion)
1.2 Requisitos de Compliance
•	WCAG 2.1 AA: Contraste mínimo 4.5:1 para texto, 3:1 para elementos grandes
•	RETECH: Cumplimiento de requisitos técnicos para financiación institucional
•	Junta de Andalucía: Manual de identidad corporativa para programas públicos
•	RGPD: Interfaces que faciliten el ejercicio de derechos de datos

7. Consideraciones de Accesibilidad WCAG
7.1 Requisitos Obligatorios para Verticales Institucionales
•	Contraste mínimo 4.5:1: Para texto normal contra fondo
•	Contraste mínimo 3:1: Para texto grande (18px+) y elementos UI
•	Focus visible: Todos los elementos interactivos con outline visible
•	Tamaño mínimo de toque: 44x44px para móvil
•	prefers-reduced-motion: Animaciones desactivables
•	Navegación por teclado: Tab order lógico, skip links
7.2 CSS para Accesibilidad
/* Respetar preferencias de movimiento reducido */
@media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}

/* Focus visible mejorado */
:focus-visible {
  outline: 3px solid var(--color-accent);
  outline-offset: 2px;
}

/* Skip link para navegación */
.skip-link {
  position: absolute;
  top: -40px;
  left: 0;
  padding: 8px 16px;
  background: var(--color-primary);
  color: white;
  z-index: 100;
}
.skip-link:focus {
  top: 0;
}
