## Slide 3

M5-1  │  Packs Listos para Vender

Tu catálogo digital: tu tienda 24/7
Cada pack tiene ficha profesional: nombre, descripción, entregables, precio, condiciones
El catálogo tiene URL pública: la compartes con cualquier potencial cliente
Botón «Contratar este servicio» → formulario de contacto o pago directo
Cobro recurrente mensual vía Stripe: tu cliente paga, tú cobras automáticamente
Si no tienes un catálogo publicado, no tienes un negocio. Hoy lo solucionamos.
